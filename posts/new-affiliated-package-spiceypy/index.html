New Affiliated Package: SpiceyPy

PlanetaryPy Technical Committee

2021-01-27 11:10

The SpiceyPy is a Python wrapper for the SPICE Toolkit. This package has been used in the planetary sciences community since 2016, and we are proud to have it listed as a PlanetaryPy Affiliated Package.

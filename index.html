PlanetaryPy

Python tools for planetary science

PlanetaryPy Overview

Planetarypy is an open source toolkit for planetary image and data processing written in ython.

Information and links to our current projects are provided below.

PlanetaryPy Projects

Planetaryimage

Planetaryimage is a python PDS and Isis Cube file parser that handles processing for planetary data.

Refer to the project's documentation for additional information.

PDSview

PDSview is a python PDS Image Viewer that provides open source access planetary imagery.

Refer to the project's documentation for additional information.

pvl

pvl provides python implementation of PVL (Parameter Value Language) to enable open source access to planetary data.

Refer to the project's documentation for additional information.

PyStamps

PyStamps is a python PDS image thumnbail browser provides open source access to planetary data.

Refer to the project's documentation for additional information.

Authors and Contributors

In 2015, Austin Godber (@godber), Bhavin Nayak (@bvnayak), Nikhil Kalige (@nikhilkalige), TJ Slezak (@tjslezak), and Perry Vargas (@pbvarga1) founded PlanetaryPy as a black operation to provide a toolkit supporting all planetary datasets under the guise of upgrading standards for the upcoming Mastcam-Z instrument on the Mars 2020 rover.

If you'd like to help support open access and availability of planetary mission data your contributions are welcomed. This project is a collaborative effort consisting of current and former students at Arizona State University, the USGS Astrogeology Branch, and NASA affiliates. Our motivation is to provide open source access and ease of use for all planetary datasets.

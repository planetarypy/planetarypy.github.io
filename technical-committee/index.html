Technical Committee

The PlanetaryPy Technical Committee (TC) is the governing body for the PlanetaryPy Project. All of their work is done in the open via the TC GitHub repo. To learn more about what the TC is, how it operates, or to get involved, you can read its Charter.

Meetings

The PlanetaryPy TC has regular meetings. All are welcome to attend.

Agendas for upcoming meetings, and notes from previous meetings can be found in the TC's repo at https://github.com/planetarypy/TC/tree/master/Meetings.
